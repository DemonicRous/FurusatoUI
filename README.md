# Furusato UI

Современные интерфейсы новых версий Minecraft, заново реализованные для
Minecraft Forge 1.12.2.

Проект использует Minecraft 1.20 как визуальный и поведенческий ориентир, но не
содержит опубликованного декомпилированного кода Minecraft или MCP-Reborn.

## Требования

- JDK 8
- Minecraft 1.12.2
- Forge 14.23.5.2847 или новее для 1.12.2
- Furusato Core 0.2.0 или новее

## Реализовано

- замена стандартного главного меню;
- текстурированный компонент кнопки на основе стандартного `widgets.png`;
- состояния normal, hover, disabled и keyboard focus;
- навигация по меню клавишами Tab/стрелками и активация Enter/Space;
- переходы к одиночной игре, мультиплееру, списку модов, настройкам и языкам;
- возможность отключить новое меню в `config/furusatoui.cfg`.

## Сборка

```powershell
.\gradlew.bat build
```

Готовый JAR создаётся в `build/libs`.

## Запуск среды разработки

Репозитории `FurusatoCore` и `FurusatoUI` должны находиться рядом. Команда

```powershell
.\gradlew.bat runClient
```

автоматически соберёт Furusato Core и скопирует основной JAR в `run/mods` перед
запуском клиента. В обычной сборке Minecraft оба JAR необходимо положить в
папку `mods` вручную.
